# 🏠 Nandagokula Boys PG Website

A modern and responsive website built for **Nandagokula Boys PG (Mangalore)** to showcase facilities, location, and contact details for students and working professionals.

🔗 **Live Website:** https://nandagokulapg.com/  
📂 **GitHub Repository:** https://github.com/SooryaBhat/Nandagokula_PG.git  

---

## 📌 About the Project

This project is a simple business website designed to help users:

- Explore PG facilities and rooms  
- View images of the property  
- Find location easily using Google Maps  
- Contact the PG owner quickly  

The goal of this project is to provide an **online presence** for a local PG so that people can discover it through search engines and maps.

---

## 🚀 Features

- ✅ Clean and modern UI design  
- ✅ Fully responsive (works on mobile, tablet, desktop)

---

## 🌐 Deployment

The website is deployed using **Netlify** and connected with a custom domain.

### Steps followed:
1. Created static website using HTML, CSS, JS  
2. Uploaded code to GitHub  
3. Connected GitHub repo to Netlify  
4. Added custom domain (nandagokulapg.com)  
5. Configured DNS and SSL (HTTPS)
